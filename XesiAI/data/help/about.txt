About
-----
XesiAI.exe allows you to detect and recognize faces, texts, and objects. User can either use the local computer or an Arduino module to input data.
It supports real-time face detection, face recognition and object recognition. It is built using Java on the Spring Boot framework with the help of
OpenCV (4.5.1) library.

Prerequisites for XesiAI.exe
----------------------------
1. Minimum version of JRE: 1.7
2. OS: Windows
